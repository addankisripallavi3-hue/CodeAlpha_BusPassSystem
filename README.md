# CodeAlpha_BusPassSystem
# Cloud-Based Bus Pass System

A simple Java-based online bus pass booking system developed as part of the **CodeAlpha Internship Task 3**. The system demonstrates cloud computing concepts such as secure ticket storage, scalability, and reliability.

#Features:
* Online bus pass booking
* Prevents ticket loss and theft
* Correct ticket pricing
* Simulated cloud storage
* Reliable and scalable design

#Technologies Used:
* Java
* Eclipse IDE
* Object-Oriented Programming (OOP)
* Cloud Computing Concepts

#Project Structure:
CodeAlpha_BusPassSystem
│
├── BusPass.java
├── TicketPrice.java
├── CloudStorage.java
└── BusPassSystem.java

#How It Works:
1. User enters passenger details.
2. Ticket price is calculated.
3. Ticket information is stored securely.
4. A digital bus pass is generated.
5. Booking confirmation is displayed.
